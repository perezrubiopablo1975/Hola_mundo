# Primer Repositorio

Este repositorio ha sido creado como parte de una tarea del módulo de programación en IA del curso IABD. El objetivo es familiarizarme el uso de herramientas de versionado como GitHub, aprendiendo a crear un repositorio, añadir archivos y como documentar el proyecto. He tenido que ver otros prooyectos y documentarme sobre ello, pues nunca lo había hecho.

## Contenido

- `README.md`: Este archivo, escrito en lenguaje de marcación, contiene una breve explicación de la tarea. También he tenido que entender que era lenguaje de marcación.
- `hola_mundo.py`: Un archivo sencillo en Python que muestra por consola "Hola Mundo".

## Objetivo de la tarea

- Crear tu primer repositorio.
- Añadir mi archivo en Python.
- Añadir un archivo README con una explicación básica.
